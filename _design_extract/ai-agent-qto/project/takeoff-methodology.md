## 1. งานโครงสร้าง (Structural) ✅ ทำแล้ว
แยก 4 sub-group: **ฐานราก → เสา → คาน → พื้น**
แต่ละ group มี: คอนกรีต · แบบหล่อ · เหล็กเสริม (แยกเบอร์) · ลวดผูกเหล็ก · ตะปู
**Lean Concrete fc=140** เฉพาะงานที่ติดดิน (ฐานราก + พื้นชั้น 1)

## 2. งานสถาปัตยกรรม (Architectural)
แยก 6 sub-group:

### 2.1 งานฝ้าเพดาน (Ceiling) — หน่วย ตร.ม.
- C1 ยิปซั่ม 9mm + โครงเหล็กชุบสังกะสี + ทาสี
- C2 ยิปซั่ม 9mm กันชื้น
- C3 Aluminium Cell 5×5 ซม.
- C4 อะคูสติก ระบบซ่อนโครง
- C5 ฉาบเรียบทาสี (พื้นที่ คสล. เดิม)

### 2.2 งานผนัง (Wall) — หน่วย ตร.ม.
- P1 ก่ออิฐมอญครึ่งแผ่น + ฉาบเรียบทาสีอะคริลิค
- P2 ก่ออิฐมอญ + กรุกระเบื้อง 60×60
- P3 ผนัง คสล. กรุกระเบื้อง
- P4 ผนัง คสล. ฉาบเรียบทาสี
- P6 กรุหินแกรนิต
- P7 ยิปซั่มบอร์ด 12mm + โครงเคร่า

### 2.3 งานพื้น (Floor) — หน่วย ตร.ม.
- F1 กระเบื้องพอร์ซเลน 60×60
- F2 ปรับระดับ+กันซึม+กระเบื้อง (ห้องน้ำ)
- F3 หินแกรนิต
- F5 Epoxy Coating Self Leveling
- F8 คสล.+กันซึม Polyurethane (ดาดฟ้า)
- บัวพื้น (เส้นตรง) — ม.

### 2.4 งานประตู-หน้าต่าง — หน่วย ชุด
- ประตู: หนีไฟ, WPC (1เปิด/2เปิด), อลูมิเนียม, เหล็ก, ไม้
- หน้าต่าง: บานกระทุ้ง, ติดตาย, เลื่อน, เกร็ด (อลูมิเนียม)

### 2.5 งานหลังคา (Roof)
- หลังคาแผ่นเหล็ก + ฉนวน PU — ตร.ม.
- ครอบสัน/ครอบข้าง — ม.
- เชิงชาย — ม.
- สกรูยึด — ตัว

### 2.6 งานเบ็ดเตล็ด (Misc)
- จมูกบันได, ราวบันได — ม.
- ตัวอักษรสแตนเลส — ชุด

**เกณฑ์เผื่อ:**
- อิฐมอญ +3% (1 ตร.ม. ครึ่งแผ่น = 70 ก้อน)
- กระเบื้อง +5%
- สี 10 ตร.ม./ลิตร × 2 รอบ
- ฉาบปูน 15mm หนา = 0.015 ลบ.ม./ตร.ม.

**Markup:** ค่าแรง 25-35% ของวัสดุ

## 3. งานไฟฟ้า (Electrical)
แยก 8 sub-group:

### 3.1 สายไฟ + ท่อร้อยสาย (Wiring + Conduit) — หน่วย ม.
แยกขนาดและชนิด:
- THW 1.5 / 2.5 / 4 / 6 / 10 / 16 / 25 sq.mm
- NYY (สายเมน)
- EMT/IMC/RSC/PVC Conduit ½" / ¾" / 1" / 1¼" / 1½" / 2"
- เผื่อแนวดิ่ง 10% ของแนวนอน

### 3.2 ดวงไฟ (Lighting) — หน่วย ดวง
แยกชนิด:
- LED Downlight 9W / 18W
- LED Panel 36W / 60W
- LED Tube T8 18W
- Outdoor Floodlight LED 100W

### 3.3 เต้ารับ + สวิตช์ (Outlets & Switches) — หน่วย ตัว
- Duplex Outlet 16A
- Single Outlet 16A WP
- Switch 1-way / 2-way / 3-way

### 3.4 ตู้ MDB + Sub Panel — หน่วย ใบ
- MDB
- Sub-Panel ต่อชั้น
- Breaker MCB / MCCB

### 3.5 Fire Alarm
- FCP — ตู้
- Smoke Detector — จุด
- Heat Detector — จุด
- Manual Station — จุด
- Bell — จุด

### 3.6 CCTV + ระบบสื่อสาร
- CCTV Camera — กล้อง
- NVR — เครื่อง
- Telephone Outlet — จุด
- Data Outlet (Cat6) — จุด

### 3.7 Lightning Protection
- หัวล่อฟ้า — ชุด
- สายตัวนำลง Cu — ม.
- หลักดิน — หลัก

### 3.8 Generator + ระบบสำรอง
- เครื่องกำเนิดไฟฟ้า — ชุด
- ATS (Auto Transfer Switch) — ชุด
- UPS — ชุด

**Markup:** ค่าแรง 30% + ค่าทดสอบ 5%

## 5. งานดับเพลิง (Fire Protection)

### 5.1 ระบบ Sprinkler — ตามมาตรฐาน NFPA 13
- ท่อ Schedule 40 Black Steel — ม. แยกขนาด (1", 1¼", 1½", 2", 2½", 3", 4", 6")
- ข้อต่อ Grooved / Threaded — อัน
- หัวสปริงเกอร์ — หัว แยกชนิด

  - Upright 155°F
  - Ceiling Conceal 155°F
  - Sidewall

- ตู้สายฉีดน้ำดับเพลิง (FHC) — ตู้
- หัวจ่ายน้ำ Fire Hydrant — หัว
- FDC (Fire Dept Connection) — ชุด

### 5.3 อุปกรณ์ Pump Room
- Electric Fire Pump — ชุด
- Diesel Fire Pump — ชุด
- Jockey Pump — ชุด
- Pressure Tank — ชุด

### 5.4 ระบบดับเพลิงพิเศษ
- ระบบ FM200 / Novec — ระบบ
- ระบบ CO2 — ระบบ
- ถังดับเพลิงพกพา — ถัง

### 5.5 อุปกรณ์เบ็ดเตล็ด
- Alarm Valve — ชุด
- Roof Manifold — ชุด
- Water Hammer Arrester — ชุด

**Markup:** ค่าแรง 30% + ค่าทดสอบ 5% (Hydrostatic Test)

## 7. งานภูมิสถาปัตย์ (Landscape)
แยก 4 sub-group:

### 7.1 Hardscape — ตร.ม.
- ทางเดินหินแกรนิตธรรมชาติ
- ลานคอนกรีตพิมพ์ลาย
- ลานหินแกรนิตชั้นนอก

### 7.2 Softscape — ตร.ม. / ต้น
- พื้นที่ปลูกหญ้านวลน้อย + Sprinkler — ตร.ม.
- ต้นไม้ใหญ่ประดับ (สูง 3-5 ม.) — ต้น
- ไม้พุ่ม / ไม้คลุมดิน — ต้น

### 7.3 Hardscape Detail
- กระบะปลูกไม้พุ่ม คสล. กรุหิน — ม.
- ขอบสนาม คสล. — ม.
- น้ำพุ / ลาน Decor — เหมา

### 7.4 ระบบ Irrigation
- ท่อ PE 1" — ม.
- หัวสปริงเกอร์ Pop-up — หัว
- Controller + Solenoid Valve — ชุด

## หลักการร่วม (Universal Rules)

### Factor F (กรมบัญชีกลาง 2560)
| ส่วนประกอบ | สัดส่วน |
|---|---|
| ค่าวัสดุ | 100% (Base) |
| ค่าแรงงาน | 25-35% |
| ค่าดำเนินการ (Overhead) | 10% |
| กำไร | 8-10% |
| ภาษีมูลค่าเพิ่ม | 7% |

### หน่วยมาตรฐาน
| ลักษณะงาน | หน่วย |
|---|---|
| งานพื้นที่ (ฝ้า/ผนัง/พื้น/กันซึม) | ตร.ม. |
| งานปริมาตร (คอนกรีต) | ลบ.ม. |
| งานยาว (ท่อ/สายไฟ/ราว/บัว) | ม. |
| งานนับชิ้น (สุขภัณฑ์/ดวงไฟ/วาล์ว) | ชุด / ตัว / อัน / หัว |
| งานน้ำหนัก (เหล็กเสริม/ลวด/ตะปู) | กก. |
| งานต้น | ต้น |
| งานเหมา (Markup, T&B) | งวด / เหมา |

### เผื่อ Waste มาตรฐาน
- คอนกรีต: +3%
- เหล็กเสริม: +5-10%
- อิฐมอญ: +3%
- กระเบื้อง: +5%
- สี: +10% (ทา 2 รอบ)
- ปูนทราย: +5%
- ท่อ PVC: +5% (เผื่อตัด)
- สายไฟ: +10% (เผื่อต่อ)
