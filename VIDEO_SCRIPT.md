## 🎨 Style Guide สำหรับ Animation

### 🎨 สีหลัก
| สี | Hex | ใช้กับ |
|----|-----|--------|
| Navy | `#1F3A5F` | Primary background, text |
| Steel | `#3A6EA5` | Secondary, gradients |
| Green | `#2DBE60` | CTA, success, highlights |
| White | `#FFFFFF` | Background, text |

### 🖋️ ฟอนต์
- **Heading:** Prompt (Bold 600-700)
- **Body:** Sarabun (Regular 400-500)

### ✨ Animation Style
- **Easing:** `cubic-bezier(0.4, 0, 0.2, 1)` (smooth)
- **Duration:** Standard 0.2-0.4s
- **Modal/Panel:** Slide-in 0.25s
- **Hover:** TranslateY(-2px) + shadow

### 🎬 Camera Movement
- ✅ Smooth zooms (1.0 → 1.1)
- ✅ Pan ช้าๆ (no shake)
- ✅ Cut sharp ไม่ใส่ transition
- ❌ ห้ามใช้ flip / rotate / cube transitions

## 🚀 Production Tips

### 🎯 Best Practices
1. **Hook ใน 5 วินาทีแรก** — ดึงดูดให้คนดูต่อ
2. **Show, don't tell** — ให้ผู้ชมเห็น UI จริง ไม่ใช่อธิบายอย่างเดียว
3. **Cursor animation** — Mouse movement ต้อง smooth ไม่กระชาก
4. **Highlight ของสำคัญ** — ใช้ glow / circle ชี้
5. **Subtitle ภาษาไทย** — เผื่อปิดเสียง

### 🎤 Recording Tips
1. ใช้ Microphone คุณภาพ (ไม่เอา onboard mic)
2. ห้อง quiet ไม่มี echo
3. พูดช้าๆ ชัดๆ
4. Pause ระหว่าง scene 1-2 วินาที (เผื่อตัดต่อ)
