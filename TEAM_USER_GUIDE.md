## 1️⃣ การสมัครสมาชิก (Sign Up)

### Step 1: เปิดเว็บไซต์
1. เปิดเบราว์เซอร์ (แนะนำ Chrome หรือ Edge)
2. พิมพ์ URL: **https://teamcm-doc-agent.vercel.app**
3. กด Enter

### Step 2: คลิก "สมัครสมาชิก"
- ที่หน้า Login → คลิก tab **"สมัครสมาชิก"**

### Step 3: กรอกข้อมูล
| ช่อง | ตัวอย่าง |
|------|---------|
| **ชื่อ-นามสกุล** | ชื่อจริงของคุณ เช่น `สมชาย ใจดี` |
| **Email** | Email บริษัท เช่น `Somchai_J@teamcm.co.th` |
| **Password** | รหัสผ่านอย่างน้อย 6 ตัว |

> 💡 **เก็บรหัสไว้ในที่ปลอดภัย** — ใช้ Password Manager หรือ Browser save

### Step 4: คลิก "สมัครสมาชิก" → รอข้อความ ✅
```
✅ สมัครสำเร็จ! กรุณายืนยัน Email ที่ส่งให้คุณ
```

### Step 5: ยืนยัน Email
1. เปิด **Outlook / Gmail / Email** ของคุณ
2. หา email จาก **Supabase** (subject: "Confirm your email")
3. คลิกปุ่ม **"Confirm your email"** ในอีเมล
4. หน้าใหม่จะเปิด → กลับมาที่เว็บได้เลย

> ⚠️ **หาอีเมลไม่เจอ?**
> - ตรวจที่ **Spam / Junk folder**
> - รอ 2-3 นาที (บางครั้งช้า)
> - ลองสมัครใหม่ (ถ้ายังไม่ confirm)

## 5️⃣ อัปโหลดเอกสาร

### Step 1: คลิก "อัปโหลด"
- ที่ Topbar → คลิก **"📤 อัปโหลด"**
- หรือคลิก **"เริ่มอัปโหลดเอกสาร"** ที่ Hero
- หรือคลิก **"+ เพิ่มเอกสาร"** ที่ section ภาพรวม

### Step 2: เลือกโครงการ
- เลือกโครงการจาก dropdown

### Step 3: เลือกประเภทเอกสาร

**🤖 จัดประเภทอัตโนมัติ** (แนะนำ) — ระบบจะตรวจจากชื่อไฟล์

**หรือเลือกเอง:**

| ประเภท | 🎨 สี | ตัวอย่างไฟล์ |
|--------|------|------------|
| 🟢 **EIA** | เขียว | EIA_MRT-PP.pdf |
| 🔵 **แบบ** | น้ำเงิน | Drawing_Floor1.dwg |
| 🟣 **สัญญา** | ม่วง | Contract_2025.pdf |
| 🟠 **แรงงาน** | ส้ม | Labor_Safety.pdf |
| 🔷 **มาตรฐาน** | ฟ้า | มยผ-1234.pdf |
| 🌸 **MOM** | ชมพู | Meeting_Minutes.pdf |
| 🔴 **ราคา/BOQ** | แดง | BOQ_Project.xlsx |
| ⚫ **อื่นๆ** | เทา | Other.pdf |

### Step 4: ลากไฟล์มาวาง (Drag & Drop)

**วิธีที่ 1: ลาก** — ลากไฟล์จาก File Explorer มาวางในกรอบประ
**วิธีที่ 2: คลิก** — คลิกกรอบประ → เลือกไฟล์

✅ รองรับ: PDF, Word, Excel, JPG, PNG (หลายไฟล์พร้อมกันได้)

### Step 5: ดู Progress Bar
- แต่ละไฟล์มี progress bar เขียวๆ
- ระบบ **detect version อัตโนมัติ** (เช่น Rev1, Rev2, v3)
- เมื่อขึ้น ✅ = พร้อม upload

### Step 6: คลิก "ยืนยันอัปโหลด"
✅ ไฟล์ถูกบันทึกในระบบ + แจ้งเตือนสีเขียว

## 6️⃣ ดูเอกสารในโครงการ

### Step 1: คลิกที่โครงการ
- ใน "โครงการล่าสุด" → คลิกที่ card โครงการ

### Step 2: Drawer เปิดขึ้นจากขวา
แสดง:
- **Header:** ชื่อโครงการ + รหัส + เจ้าของงาน + จำนวนไฟล์ + ขนาดรวม
- **ปุ่ม:** Zip ทั้งโครงการ / Zip เฉพาะล่าสุด / ลบโครงการ
- **Search:** ค้นหาในเอกสาร
- **Filter:** ✨ "ล่าสุด" + ประเภทเอกสาร (chips)
- **รายการไฟล์:** จัดกลุ่มตาม baseName + แสดงทุก Version

### 🔍 ค้นหาไฟล์
- พิมพ์คำสำคัญใน search box
- กรองตามประเภท (คลิก chip)
- เปิด **"ล่าสุด"** เพื่อดูเฉพาะ Version ล่าสุด

### 📊 Version Groups
แต่ละกลุ่มเอกสารแสดงทุก Version:
```
[EIA] EIA_MRT-PP-เตาปูน  (3 versions)
  📄 EIA_MRT-PP-เตาปูน_Rev3.pdf  [ล่าสุด]
  📄 EIA_MRT-PP-เตาปูน_Rev2.pdf  [เก่า]
  📄 EIA_MRT-PP-เตาปูน_Rev1.pdf  [เก่า]
```

### 🎯 ทำอะไรกับไฟล์ได้

| ปุ่ม | หน้าที่ |
|------|---------|
| 👁️ | ดูตัวอย่าง (Preview) |
| ⬇️ | ดาวน์โหลด |
| 🗑️ | ลบไฟล์ (มี Confirm) |

## 9️⃣ Tips & Best Practices

### ✅ ตั้งชื่อไฟล์ให้ดี

**👍 ดี:**
- `EIA_MRT-PP_Rev3.pdf` — ระบุประเภท + โครงการ + version
- `BOQ_ABC-Tower_v2.xlsx`
- `MOM_meeting18_2025-06-01.pdf`

**👎 ไม่ดี:**
- `document.pdf` — ไม่บอกอะไรเลย
- `file123.docx` — ไม่ตรวจ version ไม่ได้
- `เอกสารใหม่.pdf` — ทั่วไปเกินไป

### ✅ ใช้ Auto-detect ประเภท
ระบบจะตรวจจากชื่อไฟล์อัตโนมัติ ถ้าตั้งชื่อดี เช่น:
- `eia_xxx.pdf` → จัดเป็น EIA
- `drawing_xxx.dwg` → จัดเป็น แบบ
- `contract_xxx.pdf` → จัดเป็น สัญญา

### ✅ อัปโหลด Version ใหม่
- ใช้ baseName เดิม แต่เพิ่ม Rev/v
- เช่น Rev1 → Rev2 → Rev3
- ระบบจะ mark เฉพาะอันใหม่สุดว่า "ล่าสุด" อัตโนมัติ

### ✅ Search ก่อนถาม AI
- ถ้ารู้ชื่อไฟล์ → ใช้ Search ใน Drawer (เร็วกว่า)
- ถ้าค้นหาแบบ "ขอเอกสาร X" → ใช้ AI Chat

### ⚠️ ข้อจำกัด
- ไฟล์ขนาดสูงสุด: **100 MB / ไฟล์**
- พื้นที่รวม: **1 GB / ทีม** (Free tier)
- จะแจ้งเมื่อใกล้เต็ม

## 🆘 ปัญหาที่พบบ่อย (FAQ)

### Q1: ลืมรหัสผ่าน
**A:** ติดต่อ Admin: `Nitid_S@teamcm.co.th` เพื่อ Reset

### Q2: Upload ล้มเหลว "RLS error"
**A:** Refresh หน้าเว็บ → Login ใหม่ → ลองอีกครั้ง

### Q3: ไฟล์ไม่ขึ้นใน Dashboard
**A:**
1. กด **F5** เพื่อ Refresh
2. ตรวจว่าเลือกโครงการถูกไหม
3. Filter "ล่าสุด" เปิดอยู่หรือไม่

### Q4: AI Chat ไม่ตอบ
**A:**
1. Refresh หน้าเว็บ
2. ลองพิมพ์คำถามใหม่ที่ชัดเจน
3. ถ้ายังไม่ได้ → แจ้ง Admin

### Q5: Email confirmation ไม่ส่ง
**A:**
1. รอ 2-3 นาที
2. เช็ค Spam folder
3. ลองสมัครใหม่ด้วย email อื่น

### Q6: Login ไม่ได้บน Mobile
**A:** ระบบรองรับ Mobile แต่ใช้บน Desktop จะสะดวกกว่า

## 🎯 Workflow แนะนำสำหรับทีม

1. สร้างโครงการ → ตั้งรหัส + สี + เจ้าของงาน

### 📌 สำหรับ Document Officer
1. ตรวจไฟล์ใหม่ใน "กิจกรรมล่าสุด"
2. ตรวจ Auto-detect ถูกประเภทไหม
